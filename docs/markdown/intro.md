# Markdown Cheat Sheet

## Headers
- `# H1`
- `## H2`
- `### H3`
- `#### H4`
- `##### H5`
- `###### H6`

## Emphasis
- `*italic*` or `_italic_`
- `**bold**` or `__bold__`
- `**_both_**`
- `~~strikethrough~~`

## Lists
### Ordered List
1. First item
2. Second item
3. Third item

### Unordered List
- First item
- Second item
- Third item

## Links
- `[title](https://)`
- `[title](link "Hover Text")`

## Images
- `![alt text](image.jpg)`
- `![alt text](image.jpg "Title Text")`

## Code
- Inline `code`
- ```
  Block code
  ```

## Blockquotes
> Blockquote

## Horizontal Rule
- `---` or `___`

## Table
| Syntax | Description |
| ----------- | ----------- |
| Header | Title |
| Paragraph | Text |

## Escape Characters
- `\` Backslash
- `\`` Backtick
- `\*` Asterisk
- `\_` Underscore
- `\{\}` Curly Braces
- `\[\]` Square Brackets
- `\(\)` Parentheses
- `\#` Hash Mark
- `\+` Plus Sign
- `\.` Dot
- `\!` Exclamation Mark

## Task Lists
- `- [x] Checked item`
- `- [ ] Unchecked item`
